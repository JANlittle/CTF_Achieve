1、验证规则列表：array(字段,验证规则,错误提示,[验证条件,附加规则,验证时间])
例如：array(array('field','require','标题必须填写！'),array('field','3,10','标题必须是3到10位！',0,'length',3)),...
附加规则 说明
regex 正则验证（默认），基本规则：'/^(.*+)$/'、require、email、url、currency 货币、number
function 函数验证，格式：格式：myfunction
callback 回调函数验证，当前模型类的一个方法，格式：callbackfun
confirm 验证表单中的两个字段是否相同，格式：password
equal 验证是否等于某个值，格式：100
notequal 验证是否不等于某个值，格式：100
in 验证是否在某个范围内，格式：100,200或者array(100,200)
notin 验证是否不在某个范围内，格式：100,200或者array(100,200)
length 验证长度，格式：5或者5,10
between 验证范围，格式：1,31或者array(1,31)
notbetween 验证不在某个范围，格式：100,200或者array(100,200)
expire 验证是否在有效期，格式：2012-1-15,2013-1-15，也可以使用时间戳定义
ip_allow 允许的IP地址列表，格式：201.12.2.5,201.12.2.6
ip_deny 禁止的ip地址列表，格式：201.12.2.5,201.12.2.6
unique 验证是否唯一，格式：array('name','','帐号名称已经存在！',0,'unique',1)
2、验证条件列表
条件 说明
self::EXISTS_VALIDATE 或者0 存在字段就验证（默认）
self::MUST_VALIDATE 或者1 必须验证
self::VALUE_VALIDATE或者2 值不为空的时候验证
3、验证时间列表
验证时间 说明
self::MODEL_BOTH或者3 全部情况下验证（默认）
self::MODEL_INSERT或者1 新增数据时候验证
self::MODEL_UPDATE或者2 编辑数据时候验证
1、验证规则列表：array(完成字段,完成规则,[完成时间,附加规则])
例如：array(array('field','md5',1,'function'),array('field','sha1',1,'function')),...
附加规则 说明
function 使用函数，表示填充的内容是一个函数名，格式：md5
callback 回调方法 ，当前模型类的一个方法，格式：callbackfun
field 用其它字段填充，表示填充的内容是一个其他字段的值，格式：email
ignore 为空则忽略
2、完成时间
设置 说明
self::MODEL_INSERT或者1 新增数据的时候处理（默认）
self::MODEL_UPDATE或者2 更新数据的时候处理
self::MODEL_BOTH或者3 所有情况都进行处理
1、基础字段定义
字段类型 说明
TINYINT 有符号的表示范围是-128到127,无符号的表示范围是0到255
SMALLINT 有符号的表示范围是-32768到32767,无符号的表示范围是0到65535
MEDIUMINT 有符号的表示范围是-8388608到8388607,无符号的表示范围是0到16777215
INT 有符号的表示范围是-2147483648到2147483647,无符号的表示范围是0到4294967295。
FLOAT 小型浮点数
DOUBLE 双精度浮点数
VARCHAR 可变长（0-65,535）字符串，实际可用最长长度由最长字段长度定义
TEXT 一个最长65536字符的TEXT字段，采用一个前缀（两个字节）表示当前值的字节长度
TINYTEXT 一个最长255字符的TEXT字段，并采用一个前缀（一个字节长）来表示当前值的字节长度
MEDIUMTEXT 一个最长16777215字节的TEXT字段，在字节开始处用三个字节保存当前值的长度
2、参数规则
规则名称 说明
布尔(bool) 格式：开启:1|关闭:0，布尔值请保证值只有1跟0，1-是，0-否
密码(password) 格式：check:1|0，1-开启(请配置自动验证规则：array(array('field','field_checkpassword','密码确认不正确！',0,'confirm',3))，请保证field_checkpassword保持不变，0-关闭(请删除自动验证规则)
枚举(select) 格式：否:0|是:1
单选(radio) 格式：画画:1|游泳:2|写作:3
多选(checkbox) 格式：推荐:command|头条:header|幻灯:slide
联动菜单(linkagemenu) 格式：array(1)，其中1代表联动菜单的id
缩略图(thumb|picture|file) 格式：array(200,150,3072,'single','uploadfile')，宽度：200，高度：150，上传大小：单位KB，上传模式：single(单)|multiple(多)，文件保存目录：uploadfile，不填则取系统值
